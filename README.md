# Gradient Boosting for Regression and Classification

## Description
This project demonstrates the implementation of **Gradient Boosting** models for both **regression and classification tasks**. Gradient Boosting is an ensemble learning technique that builds models sequentially, where each new model learns from the errors of previous models to improve prediction accuracy.

---

## Project Overview
Gradient Boosting combines multiple weak learners (usually decision trees) to create a strong predictive model. Each model is trained on the **residual errors** of the previous model, gradually improving performance.
